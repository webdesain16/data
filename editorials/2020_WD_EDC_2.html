Kita memiliki dua kasus penting, yaitu :

  1. Ketika titik berada di luar kertas

    Pada kasus ini, cek apakah titik berada dalam kertas tersebut. Jika dan dan dan langsung outputkan Di Luar Jangkauan karena titik tersebut berada diluar kertas dan program berhenti sampai sini.

  2. Ketika titik berada didalam kertas

Pada kasus ini, nilai dan berada diantara dan eksklusif. Observasi gambar diatas, maka kita tahu bahwa urutan sumbu yang kita simpan atau outputkan adalah : 1) sumbu- negatif, sumbu- positif 2) sumbu- positif, sumbu- positif 3) sumbu- negatif, sumbu- negatif 4) sumbu- positif, sumbu- negatif

Maka untuk kasus ini, cukup lakukan dua nested loop dengan loop terluar menyatakan sumbu- yang sedang kita jelajahi dan loop yang dalam menyatakan sumbu- yang sedang kita jelajahi.

Ada 3 kasus saat kita sedang menjelajahi kedua sumbu, yaitu:

  1. Titik yang kita jelajahi sama dengan titik (X,Y) maka outputkan #
  2. Titik yang kita jelajahi merupakan bagian dari sumbu maka outputkan *
  3. Selain kedua kasus diatas, outputkan (spasi)

Dan jangan lupa untuk meng-outputkan (spasi) diantara dua karakter. Berikut adalah contoh program dalam bahasa C++:

xxxxxxxxxx
   #include <bits/stdc++.h>
   using namespace std;
   int main(){
     int n,x,y;
     cin >> n >> x >> y;
     if(x >= -n && x <= n && y >= -n && y <= n){
       for(int i = n; i >= -n; i--){
         for(int j = -n; j <= n; j++){
           if(i == y && j == x) cout << "#";
           else if(i == 0 || j == 0) cout << "*";
           else cout << " ";
           if(j < n) cout << " ";
         }
         cout << endl;
       }
       cout << "( " << x << " , " << y << " )\n";
     } else cout << "Di Luar Jangkauan\n";
     return 0;
   }

Pertama, buat sebuah array 1D arr dengan tipe data boolean yang menyatakan apakah kamar ke- berisi pasien ODP/PDP. Kemudian isi array tersebut dengan false yang menyatakan tidak ada pasien ODP/PDP pada kamar ke-. Lalu lakukan looping dari sampai untuk membaca masukan yang menyatakan nomor-nomor kamar yang berisi pasien ODP/PDP, misalkan nomor kamar tersebut disimpan pada variabel rooms. Lalu ubah nilai dari arr[rooms] menjadi true.

Setelah itu, lakukan looping dari kamar nomor sampai . Apabila nomor kamar yang sedang kita jelajahi terdapat pasien ODP / PDP atau arr[i] = true, maka ubah tetangga kanan dan kirinya (apabila ada) menjadi true.

Terakhir, misalkan kita memiliki variabel ans dengan tipe data integer. Atur ans = 0. Kemudian lakukan looping dari kamar nomor 1 sampai N, apabila arr[i] = true, maka ans++. Setelah looping berhenti, cetak ans sebagai jawaban kita. Berikut adalah contoh program dalam bahasa C++:

xxxxxxxxxx
#include <bits/stdc++.h>
using namespace std;
int main(){
  bool arr[200005];
  int N,K,ans=0;
  cin >> N >>> K;
  for(int i = 1; i <= N; i++){
    arr[i] = false;
  }
  for(int i = 0; i < K; i++){
    int rooms;
    cin>>rooms;
    arr[rooms] = true;
  }
  for(int i = 1; i <= N; i++){
    if(x < N) arr[x+1] = true;
    if(x > 1) arr[x-1] = true;
  }
  for(int i = 1; i <= N; i++)
    if(arr[i] == true) ans++;
  cout << ans << "\n";
}

E. Menonton Pertandingan

Kita sediakan array 2 dimensi bernama kursi dan sebuah variabel bernama kosong dan diinisiasikan dengan nilai . Kita juga menyediakan varaibel yeyy bertipe data boolean dan diinisiasikan dengan nilai false. Variabel yeyy akan kita inisiasikan true jika Atung dan Kaka bisa menonton bersama.

Kita menggunakan nested loop untuk mengecek semua kursi. Jika kursi pada baris ke â€“ dan kolom ke â€“ bernilai , maka kita increment (menambahkan ) pada variabel kosong yang menandakan ada berapa kursi kosong di blok tersebut sekaligus kita cek apakah kursi sebelah kiri dan kanannya kosong.

Kursi sebelah kiri adalah kursi baris ke â€“ dan kolom ke . Jika kursi baris ke â€“ dan kolom ke â€“ serta kursi sebalah kirinya kosong, maka kita inisiasikan nilai true pada variabel yeyy

Untuk menentukan output, kita pertama menentukan apakah jumlah kursi yang ada di blok lebih dari atau sama dengan dua. Jika tidak, maka sudah jelas Kaka dan Atung tidak jadi nonton karena tidak ada atau hanya satu kursi kosong yang tersedia. Jika kursi kosong lebih besar atau sama dengan dua, maka kita cek lagi apakah mereka duduk bersebalahan atau tidak. Berikut adalah contoh program dalam bahasa C++:

G. Petak Pemenang

Jalan tercepat untuk mencapai petak end dari petak start adalah melalu diagonal utamanya. Jika kita observasi, jalur diagonal dari petak start menuju petak end adalah saat koordinat dari baris dan kolomnya sama

 

Soal ini dapat diselesaikan dengan nested loop (perulangan dalam perulangan) namun solusi tersebut sangatlah lama, maka kita bisa menggunakan perulangan biasa.

Pada solusi yang menggunakan nested loop, kita mengeluarkan output berupa koordinat jika nilai variabel iterasi pada for yang ada di luar sama dengan nilai variabel iterasi pada for yang ada di dalam. Berikut adalah contoh program yang mengunakan nested loop.

xxxxxxxxxx
#include <bits/stdc++.h>
using namespace std;
int main(){
  int n;
  cin>>n;
  for(int i=1; i<=n; i++){
    for(int j=1; j<=n; j++){
      if(i == j) cout << i << " "<< j << endl;
    }
  }
}
